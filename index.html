Slash The Testing Framework For Real Stuff

What is it?

Slash is a testing framework written in Python. Its primary goal is to provide a modern, extensible foundation for authoring automated tests.

Why Slash?

Designed for Beyond Unit Tests

Slash is aimed at testing real products, not math libraries. It supports many features that help with that cause -- state sharing between tests, serious customization and bundling support, proper logging management and support and much more.

Flexible

Slash doesn't get in your way. Syntax for tests is minimal, and the learning curve is moderate.

Get it

Just as easy as: 
$ pip install slash
